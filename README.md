# 🍔 Swiggy Clone App

This project is a Swiggy-inspired clone application developed for educational and demonstration purposes.
It is an independent project and is not affiliated with or endorsed by Swiggy.

---

## 📌 About the Project

This application is built to understand how modern food delivery apps work and to practice Android development by implementing real-world UI and core functionalities.

---

## 🚀 Features

* 🏠 Home Section (Browse food items)
* 🛒 Cart System (Add / Remove items)
* 👤 Profile Section
